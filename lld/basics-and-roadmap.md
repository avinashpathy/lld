# HLD vs LLD
 - HLD -> High level, system architecture, what components do we need and how do they communicate?
 - LLD -> Details of each component, how to implement individual component?

 Requirement -> HLD -> LLD -> Code Implementation

 Core Components of LLD:
 1. Classes and Objects
 2. Interfaces and Abstractions (Interface = defines contracts between components  = a set of rules (methods) that a class must follow)
 3. Relationships Between Classes: Association ("uses-a" relationship), Aggregation (weak "has-a" relationship - both can exist independently), Composition (strong "has-a" relationship), Inheritance ("is-a" relationship), Cardinality: (1:1, 1:Many, Many:Many)
 4. Method Signatures
 5. Design Patterns
